# LearnXchange - Skill Exchange Mobile Application

## 📌 Overview
LearnXchange is a Flutter-based mobile application that enables users to exchange skills through a mutual matching system. Users can offer skills they know and request skills they want to learn, allowing peer-to-peer knowledge sharing without any monetary transactions.

---

## 🎯 Objectives
- Provide a platform for free skill exchange
- Connect users with mutual learning interests
- Promote collaborative and peer-to-peer learning
- Build a simple and user-friendly mobile experience

---

## 🚀 Features

### 👤 User Management
- User registration and login (Firebase Authentication)
- Profile creation and management

### 🧠 Skill Management
- Add skills offered
- Add skills wanted
- Edit and delete skills

### 🔗 Matching System
- Rule-based matching logic
- Match users based on mutual skill exchange
- Display potential matches

### 📩 Request Management
- Send exchange requests
- Accept or reject requests
- Track request status

### 💬 Communication
- Basic chat between matched users (optional/simple implementation)

### ⭐ Rating System
- Rate users after skill exchange
- Maintain trust and quality

---

## 🛠️ Technologies Used
- **Frontend:** Flutter
- **Backend:** Firebase (Authentication, Firestore)
- **State Management:** Provider / Riverpod (optional)

## ⚙️ How It Works
1. User creates an account and logs in
2. User adds skills they offer and want to learn
3. System finds matches using rule-based logic
4. Users send and receive exchange requests
5. Once accepted, users can communicate and exchange skills
6. After completion, users can rate each other
